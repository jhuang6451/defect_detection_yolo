# 超参数调优指南 (Hyperparameter Tuning Guide)

本指南专门针对基于 YOLO 算法进行的“工业零部件小目标缺陷检测”任务编写。在您的毕业设计中，合理地调整超参数将极大地影响模型最终的精度（mAP）和收敛速度。

## 1. 学习率 (Learning Rate, LR) 与调度策略

小目标缺陷检测由于其特征少且容易受到背景噪声影响，如果初始学习率过大，可能会导致模型直接跳过最优解；但如果学习率过低，又会导致收敛缓慢甚至陷入局部最优。

### 核心建议：
* **更长的预热期 (Warmup)**：建议将 warmup epochs 从常规的 3 增加到 5 甚至 10。这能给模型充分的时间来初始化权重，防止遇到棘手的微小缺陷时产生剧烈的梯度震荡。
* **降低初始学习率 (`lr0`)**：针对小目标的细粒度特征，建议把默认学习率降低一半（例如将 `0.01` 降至 `0.005` 或更低）。
* **余弦退火 (Cosine Annealing)**：使用余弦退火策略来平滑地衰减学习率到 `lrf` (通常是 `lr0` 的 0.01 倍)，能帮助模型在训练后期更好地微调。

**示例 YAML 训练配置参考 (`custom_hyp.yaml`)**:
```yaml
lr0: 0.005          # 初始学习率 (相比默认减小)
lrf: 0.01           # 最终学习率为 lr0 * lrf (即 0.00005)
warmup_epochs: 5.0  # 预热轮数 (相比默认增大)
warmup_momentum: 0.8
warmup_bias_lr: 0.1
```

## 2. WIoU-v3 损失函数参数微调方案

您在课题中引入了 WIoU-v3 (Wise-IoU version 3) 损失网络，它的核心特点是拥有动态非单调聚焦机制，可以自动抑制质量极差的异常标注（比如工业场景中边缘极度模糊的划痕、错误标注的油污点等）。

### 核心参数解释：
* **$\alpha$ (Alpha)**: 控制非单调聚焦曲线极值点的超参数。
  * **作用**：决定了什么级别的离群样本应该受到多大的关注。
  * **微调策略**：如果您的数据集含有大量极其糟糕的错误标注或严重噪声，可以考虑略微**降低 $\alpha$ 的值（比如调到 1.8）**，使网络更早“放弃”这些害群之马。
* **$\delta$ (Delta)**: 控制降权系数斜率的力度超参数。
  * **作用**：控制抑制效果的剧烈程度。
  * **微调策略**：如果您发现训练后期 Loss 值波动非常大（震荡），并且此时 `iou_mean` 在一个较高的位置徘徊，可以尝试**增大 $\delta$**（如增至 3.5 以上），加强惩罚。

## 3. Anchor 自动聚类与 IoU 匹配阈值

诸如钢带边缘破损、狭长划痕等缺陷，其感受野的长宽比与 COCO 等自然数据集差异极大。

### 核心建议：
* **Anchor 先验聚类**：YOLO 会自动使用 K-means++ 对数据集中的真实框 (BBox) 聚类。请在终端关注输出的第一轮 anchor 生成结果。确保生成的尺寸符合长条状或者微小点状的比例。
* **放宽匹配阈值 (`iou_t`)**：小目标一旦偏离一个像素，它的 IoU 就可能断崖式下跌。在进行标签匹配 (Label Assignment) 时，默认的 `iou_t = 0.20` 可能太严格。
  * **微调策略**：建议将 `iou_t` 适度调为 `0.10 ~ 0.15`，允许更多的 Anchor 负责微小目标的预测，提高召回率。

**评估指标重点**：在使用这些参数时，多关注 `mAP@0.5` 与 `mAP@0.5:0.95` 的差值。如果 `mAP@0.5` 很高但后者很低，说明网络能发现目标，但框得不够紧凑。这时需要结合 WIoU-v3 参数作进一步平衡调整。
